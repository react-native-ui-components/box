# @react-native-ui-components/box

Box component for React Native

## Installation

```sh
npm install @react-native-ui-components/box
```

## Usage

```js
import { Box } from '@react-native-ui-components/box';

// ...

<Box>
  <Text>Hello world</Text>
</Box>;
```
